📱 WhatsApp Sender — How It Works

A simple guide for non-technical users

What Is This System?

WhatsApp Sender lets you send WhatsApp messages to many people at once — without getting banned.


Instead of sending 1000 messages from 1 number (which gets banned instantly), it spreads messages across multiple numbers, like having 7 different people sending messages.

The Problem We Solve

If you send 500 WhatsApp messages from 1 number in 1 hour, WhatsApp will ban that number within hours.


WhatsApp looks for:

  • Same message sent many times → looks like spam
  • Too many messages per hour → looks like a bot
  • No replies → looks like spam (real people get replies)
  • Same IP for multiple numbers → looks like automation

Between each message, it waits 30-90 seconds (random). This makes it look like a real person typing, not a bot.

Each number has limits. When a number hits its limit, the system skips it and uses the next one.

Warmup Period (New Numbers)

When you add a new WhatsApp number, it starts in "warmup" mode. It can only send a few messages at first, then more each day.

The system advances the warmup day automatically at midnight. You don't need to do anything.

Messages — You Control Everything

The system sends EXACTLY what you write. It does not change your words.


You have 2 options:


Option 1: One message (simple)

Hi {name}, your Lesson 1 is ready! Reply 1 to confirm.

Every person gets the same message with their name. Simple.


Option 2: Message variants (safer)

You can write up to 3 different versions of your message. The system rotates them so not everyone gets the exact same text.

Variant 1: Hi {name}, your Lesson 1 is ready! Reply 1 to confirm.
Variant 2: Hello {name}, Lesson 1 is now available! Reply 1 to confirm.
Variant 3: Salam {name}, check out Lesson 1! Reply 1 to confirm.

You write all 3. The system picks one randomly for each person. You control what they say.


Why use variants? If you send the exact same message to 500 people, WhatsApp detects it as spam. With 3 variants, it looks like a real person sending slightly different messages.

What Happens After You Start a Campaign?

  1. Sending: System sends messages one by one, rotating numbers, with random delays
  2. Tracking: Each message is logged (sent, failed, or replied)
  3. Replies: When someone replies "1", the webhook marks them as "replied" in the database
  4. Opt-out: If someone replies "STOP", they're added to a global block list — never contacted again
  5. Ban: If a number gets banned, it's removed from rotation + you get a Telegram alert
  6. Completion: When all messages are sent, you get a Telegram alert with stats
  7. Auto-resume: If not all messages were sent (capacity full), the system resumes next day automatically

Common Questions

Do I need to keep my phone on?

Yes. Each WhatsApp number needs a phone with WhatsApp installed and connected to internet. The system connects to WhatsApp through the "Linked Device" feature (like WhatsApp Web).


What happens if my phone loses internet?

The number goes offline. The system detects it and skips that number. Other numbers continue sending. When your phone reconnects, the number comes back automatically.


Can I use the same number on multiple campaigns?

Yes. The same person can receive Lesson 1 today, Lesson 2 tomorrow, Lesson 3 the day after. Each campaign is separate — sending Lesson 2 does not affect Lesson 1's status.


What if I have more recipients than capacity?

The system sends as many as it can today, then queues the rest for tomorrow. At midnight, daily counters reset and the system auto-resumes sending the remaining recipients.


How do I get new contacts into the system?

Three ways: (1) Upload a CSV file in the Campaigns page, (2) Use the webhook URL to receive contacts from your website/form, (3) Add contacts manually in the Contacts page.
