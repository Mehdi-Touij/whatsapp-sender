The Problem

Why sending from one number gets you banned

WhatsApp has strict anti-spam systems. Sending too many messages from one number triggers automatic bans.

Too many messages

Sending 100+ messages per hour from one number looks like a bot. WhatsApp bans the number within hours. The safe limit is about 20 messages per hour.

Same message repeated

If 500 people receive the exact same text, WhatsApp detects it as spam. Real people send slightly different messages each time.

No replies = spam

If you send 500 messages and nobody replies, WhatsApp thinks you are spamming. A 30%+ reply rate keeps your numbers alive.

How It Works

Four simple steps

1

Upload your contact list

Upload a CSV file with phone numbers and names. Or use the webhook URL to automatically receive contacts from your website or landing page forms. Every contact is stored in the system and can be reused across multiple campaigns.

2

Write your message

Write the exact message you want to send. The system sends it word-for-word — it never changes your text. You can use the name placeholder so each message is personalized. You are in full control of what people receive.

3

Click Start

The system sends your message to each person, one by one. It rotates between your WhatsApp numbers so no single number sends too many messages. Random delays between 30 and 90 seconds make it look like a real person typing.

4

Watch the dashboard as messages are sent. See who received, who replied, and who opted out. When someone replies "1", they are marked as confirmed. When someone replies "STOP", they are permanently removed from all future campaigns.

Warmup Period

New numbers start slow and build up over 3 days

When you add a new WhatsApp number, it cannot send 160 messages on day one. That would trigger an instant ban. Instead, it gradually increases.

Day 1 — 20 messages maximum

The number is brand new. It can only send 20 messages today. This looks like a normal person who just started using WhatsApp.

Day 2 — 60 messages maximum

The number has been active for 24 hours. The limit increases to 60. It is still being cautious but can do more work.

Day 3 — 100 messages maximum

The number is now established. It can send 100 messages. Almost at full capacity.

Day 4 and beyond — 160 messages (full capacity)

The number has graduated from warmup. It is now fully active and can send up to 160 messages per day. This happens automatically — you do not need to do anything.

Message Control

You control exactly what people receive

The system sends your message word-for-word. It does not change, rewrite, or paraphrase anything. You decide what it says.

What you write

The only thing the system replaces is {name} — it puts each person's actual name. Everything else stays exactly as you wrote it.

What they receive

Optional: Message variants for extra safety

If you want to be extra safe, you can write up to 3 slightly different versions of your message. The system rotates them so not everyone gets the exact same text. This is optional — if you write one message, the system sends that one message to everyone.

FAQ

Common questions

Do I need to keep my phone on?

Yes. Each WhatsApp number needs a phone with WhatsApp installed and connected to the internet. The system connects to WhatsApp through the "Linked Device" feature, similar to WhatsApp Web. If a phone loses internet, that number goes offline and the system skips it automatically. Other numbers continue sending.

Can I send different campaigns to the same people?

Yes. Each campaign is separate. You can send Lesson 1 on Monday, Lesson 2 on Tuesday, and Lesson 3 on Wednesday — all to the same 500 people. Sending Lesson 2 does not affect Lesson 1's data. Each campaign tracks its own sent, replied, and failed counts independently.

What happens if I have more recipients than capacity?

The system sends as many messages as it can today. The rest are queued automatically. At midnight, the daily counters reset and the system resumes sending the remaining recipients. You get a Telegram alert when the campaign auto-resumes. You do not need to restart anything manually.

How do I get contacts into the system?

Three ways. First, upload a CSV file in the Campaigns page. Second, use the webhook URL to automatically receive contacts from your website or landing page forms. Third, add contacts manually in the Contacts page. All contacts are stored permanently and can be reused across campaigns.

What happens if someone replies STOP?

The system adds their phone number to a global opt-out list. They will never receive another message from any campaign, even future ones. You get a Telegram alert when this happens. This is required by WhatsApp's anti-spam policy and protects your numbers from being reported.
